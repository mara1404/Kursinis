Naudojama papildoma biblioteka - Adafruit_Python_DHT
	Instaliavimas:

	1. Klonuoti bibliotekà ið git
		git clone https://github.com/adafruit/Adafruit_Python_DHT.git

	2. Instaliuoti Python 2.7

		sudo apt-get update
		sudo apt-get install build-essential python-dev

	3. Instaliuoti bibliotekà

		cd Adafruit_Python_DHT
		sudo python setup.py install

	
Programos paleidimas:
	1. Pereiti á src/ katalogà
	2. sudo python term.py - programa bus paleidþiama vienà kartà
